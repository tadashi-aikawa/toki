# セッションライフサイクル

セッション中は以下のタイミングで処理を実行する。

## セッション開始時

### `/task` でセッションを開始した場合

何もしない

### `/task` でセッションを開始していない場合

```
ss start "<依頼の要約>"
```

## 応答の開始前

応答中でも進捗率やフェーズに変化があれば都度実行すること。


### `/task` でセッションを開始した場合

```
ss progress <進捗率(0~1)> <フェーズ名>...
/task-progress <id>
```


### `/task` でセッションを開始していない場合

```
ss progress <進捗率(0~1)> <フェーズ名>...
```

## 応答の完了後


### `/task` でセッションを開始した場合

```
ss wait
/task-wait <id>
```

### `/task` でセッションを開始していない場合

```
ss wait
```
